# Projeto OpenGL 3D

## Dependências

- C++
- OpenGL 3.3 Core Profile
- GLFW
- GLAD
- GLM
- stb_image
- CMake

## Ambiente de Desenvolvimento

- Visual Studio Code
- MinGW-w64 (g++)
- CMake
- Windows 10

## Compilação

1. Abrir o projeto no Visual Studio Code.
2. Configurar o projeto utilizando CMake.
3. Gerar os arquivos de build.
4. Compilar o projeto.
5. Executar o executável gerado.

## Controles

### Objeto selecionado

- 1, 2, 3, 4: seleciona objeto

### Movimentação

- W: mover para cima
- S: mover para baixo
- A: mover para esquerda
- D: mover para direita

### Escala

- [: aumentar escala
- `: diminuir escala

### Rotação

- X: eixo X
- Y: eixo Y
- Z: eixo Z
- R: ativa/desativa rotação contínua

### Câmera

- ↑: avançar
- ↓: recuar
- ←: girar para esquerda
- →: girar para direita

### Luzes

- F1: liga/desliga luz 1
- F2: liga/desliga luz 2
- F3: liga/desliga luz 3

### Trajetórias

- C: adiciona checkpoint
- T: inicia/interrompe trajetória

Entrega da tarefa "Atividade vivencial 1"

- Para selecionar o objeto a ser controlado, utilize-se das teclas correspodentes aos números 1 e 2.
- Ao selecionar o objeto, aperte a tecla R para ativar a rotação nos eixos X, Y e Z.
- Para aumentar a escala dos objetos mostrados em tela, utilizar os comandos ´ e [ para, respectivamente, diminuir e aumentar.
- Utilizar as teclas X, Y e Z para rotacionar os cubos de diferentes maneiras.
- Utlizar as teclas W, A, S e D para movimentar os cubos horizontalmente e verticalmente.

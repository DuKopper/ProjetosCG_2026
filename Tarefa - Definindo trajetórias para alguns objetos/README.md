Entrega da tarefa "Definindo trajetórias para alguns objetos" 

- Utilize as teclas F1, F2 e F3 para ativar e desativar as luzes.
- Utilize as setas do teclado para movimentar a câmera em primeira pessoa.
- Para selecionar o objeto a ser controlado, utilize-se das teclas correspodentes aos números 1 e 2.
- Ao selecionar o objeto, aperte a tecla R para ativar a rotação nos eixos X, Y e Z.
- Para aumentar a escala dos objetos mostrados em tela, utilizar os comandos ´ e [ para, respectivamente, diminuir e aumentar.
- Utilizar as teclas X, Y e Z para rotacionar os cubos de diferentes maneiras.
- Utlizar as teclas W, A, S e D para movimentar os cubos horizontalmente e verticalmente.
- Utilizar a tecla C para criar checkpoints, onde o objeto selecionado irá transitar.
- Utilizar a tecla T para começar a realizar o trajeto composto pelos checkpoints criados anteriormente.

**Atenção!**
	
	- Lembre de atualizar os caminhos para os arquivos obj. e mtl. de acordo com as suas preferências.
	- Para reconfigurar os checkpoints da trajetória construida, deve ser necessário apertar a tecla ESC para fechar a aplicação
